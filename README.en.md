# Sys-Inventario

A web app to manage and visualize your personal inventory.

## Table of contents
- [Description](#description)
- [Installation](#installation)
- [Usage](#usage)
- [Tech stack](#tech-stack)

## Description
Keep your personal products organized and visible in one place.
Add, search and mark items *(still in development)*.

## Installation
1. Clone the repository
```bash
git clone https://github.com/Elias-J-Guardado/Sys-Inventario.git
```
2. Install dependencies
```bash
npm install
```
3. Start the dev server
```bash
npm run dev
```
